Project 7

Parallax Effect

I have created a parallax effect website using HTML, CSS, and JavaScript. The parallax effect is a visual technique that creates a sense of depth and movement by making the background of the webpage move at a slower pace than the foreground elements as the user scrolls up and down the page. This creates a 3D-like effect that can make designs feel more immersive and engaging.

Project 8

Modal Popup

I have created a modal popup website using HTML, CSS, and JavaScript. A modal popup is a type of window that appears on top of the webpage content, typically triggered by a button or link, and is used to display additional information, forms, or interactive elements without navigating away from the current page.
