Dicee Project

This website is a dice website that simulates the rolling of a dice using HTML, CSS, and JavaScript. The website is designed to be visually appealing and user-friendly, allowing visitors to easily roll the dice and see the results. You need to refresh the page to roll the dice of each player.

Drum Kit

This website is a drum kit website that simulates the playing of a drum kit using HTML, CSS, and JavaScript. The website is designed to be visually appealing and user-friendly, allowing visitors to easily play the drums and hear the sounds.
